# Xerenas (Arenas)

Xerena is an ERC-721 NFT that acts as a battle arena. Xerns can buy Xerenas and rent them out to gain passive income from each match that takes place on their rented out Xerenas. Each Xerena boosts skills in different ways. Xerenasâ€™ multipliers of each skill is defined by the system and it affects the overall score of a team. Xern can see which Xerena boosts which skill while purchasing one.

Example:

For Xerena 1, Rank of skills: STA-STR-INT-POW-SPD

\*In this example Intelligence (INT) hits Speed (SPD) with the multiplier of 1.1

|                    |                |
| ------------------ | -------------- |
| **SKILLS**         | **MULTIPLIER** |
| INTELLIGENCE (INT) | 1.1            |
| STRENGTH (STR)     | 1.2            |
| POWER (POW)        | 0.9            |
| SPEED (SPD)        | 0.8            |
| STRATEGY (STA)     | 1.3            |
