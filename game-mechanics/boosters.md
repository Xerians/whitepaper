# Boosters

All Xerians can maximum have 3 boosters in addition to 5 body parts and 1 card. Each booster upgrades one skill score of a particular body part.

|                  |                    |
| ---------------- | ------------------ |
| **BOOSTER**      | **SKILLS**         |
| HEAD BOOSTER     | INTELLIGENCE (INT) |
| BODY BOOSTER     | STRENGTH (STR)     |
| ARMS BOOSTER     | POWER (POW)        |
| LEGS BOOSTER     | SPEED (SPD)        |
| BACKPACK BOOSTER | STRATEGY (STA)     |
